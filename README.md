## 🎯 How to Use

### Create Account
1. Open index.html
2. Fill signup form
3. Click SIGN UP
4. See your User ID ✅
5. Switch to Sign In

### Login
1. Enter username
2. Enter password
3. Click SIGN IN
4. See your dashboard

### Manage Tasks
1. Click on section (e.g., Study 📚)
2. Type task name
3. Click "Add" or press Enter
4. Task appears instantly

### Add New Section
1. Click "Add New Section"
2. Enter section name
3. Pick an icon
4. Click Create
5. New section appears!
